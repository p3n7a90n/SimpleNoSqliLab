## Simple NoSqli Lab
## Requirements
- Node Version(v10.19.0)
- Mongodb instance

## Run the below commands to get Started after cloning the repo
```
npm install
node server.js
```
Server will start listening on port 8080.   
MongoDb Instance Url is defined in dbconfig File.
